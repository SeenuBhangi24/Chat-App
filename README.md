A full-stack Real-Time Chat Application built using the MERN Stack and Socket.io that allows users to communicate instantly through private one-to-one messaging.
The application provides 
1.secure authentication, 
2.real-time communication, 
3.message storage, and 
4.a clean responsive user interface similar to modern messaging platforms.


Real time communication flow
Frontend (React)
      ↓
Socket.io Client
      ↓
Backend (Node.js + Express)
      ↓
MongoDB Database
      ↓
Socket.io Server
      ↓
Receiver gets message instantly
